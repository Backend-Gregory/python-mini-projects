# Проект 10: Конвертер валют 💱

Консольное приложение для конвертации валют с использованием публичного API.  
Получает актуальные курсы и выполняет расчёты.

## 🎯 Что закрепляет

- Работа с API через библиотеку `requests`
- Обработка HTTP запросов и ответов
- Парсинг JSON данных
- Обработка сетевых ошибок (`ConnectionError`, `Timeout`)
- Защита пользовательского ввода
- Работа с внешними данными в реальном времени

## 📋 Функционал

### 1. Конвертация валют
- Ввод исходной валюты (например USD)
- Ввод целевой валюты (например RUB)
- Ввод суммы для конвертации
- Получение актуального курса через API
- Вывод результата с двумя цифрами после запятой

### 2. Защита от ошибок
- Проверка, что валюты не пустые
- Проверка, что сумма — положительное число
- Защита от ввода букв в сумме
- Приведение валют к верхнему регистру (регистронезависимость)

## 🛡️ Обработка ошибок

- **Ошибки сети** — нет подключения к интернету
- **Таймаут** — сервер не отвечает
- **Ошибки сервера** — статус не 200
- **Неверная валюта** — проверка наличия в ответе
- **Неизвестные ошибки** — общий `Exception`
- Понятные сообщения для каждой ситуации
