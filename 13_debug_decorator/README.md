# Проект 13: Декоратор для отладки (Logger)

Универсальный декоратор для логирования вызовов функций.  
Автоматически записывает имя функции, аргументы, результат и время выполнения.

## 🎯 Что закрепляет

- Декораторы (`@decorator`)
- Замыкания и функции высшего порядка
- Работа с `*args, **kwargs`
- Измерение времени выполнения
- Форматирование даты и времени
- Запись в файл и обработка ошибок ввода‑вывода

## 📋 Функционал

### Декоратор `@log_calls`
- **Принимает любую функцию** с любыми аргументами
- **Печатает в файл `log.txt`**:
  - дату и время вызова
  - имя функции
  - переданные аргументы
  - результат
  - время выполнения (с точностью до 4 знаков)
- **Обрабатывает ошибки** при записи в файл
- **Возвращает результат** оригинальной функции (не ломает поведение)

## 🛡️ Обработка ошибок

- **Файловые ошибки:** `IOError`, `PermissionError` — вывод предупреждения, программа не падает
- **Общий `Exception`** — ловит всё остальное, выводит сообщение

## 📝 Пример работы
```python
@log_calls
def add(a, b):
    return a + b

add(3, 5)
```
### В файле log.txt появится:
```text
Дата: 2025-03-15
Время вызова декоратора: 21:45
→ add ((3, 5), {})
← Результат: 8
⏱ Время выполнения: 0.0002 сек
```

## 🧠 Чему научился
- Писать собственные декораторы
- Передавать аргументы через *args, **kwargs
- Измерять и форматировать время выполнения
- Добавлять дату и время в логи
- Безопасно работать с файлами
- Не ломать исходную функцию
