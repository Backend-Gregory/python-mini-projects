# Проект 7: Поиск друзей в соцсети 👥

Консольное приложение для анализа дружеских связей.  
Позволяет находить общих друзей, взаимные связи и получать рекомендации.

## 🎯 Что закрепляет

- Множества (пересечения, разности, объединения)
- Словари (хранение данных пользователей)
- Операции над множествами (`&`, `-`, `|`)
- Обход словарей
- Сохранение и загрузка JSON
- Обработка ошибок ввода

## 📋 Функционал

### 1. Найти общих друзей
- Ввод двух пользователей
- Поиск пересечения их множеств друзей
- Вывод результата

### 2. Найти взаимных друзей
- Поиск всех пар пользователей, которые дружат друг с другом
- Исключение дубликатов (`user1 < user2`)

### 3. Получить рекомендации
- Сбор друзей друзей
- Исключение себя и уже существующих друзей
- Вывод потенциальных знакомых

### 4. Выход
- Завершение программы

## 🛡️ Обработка ошибок

- Защита от ввода букв в меню (`try-except`)
- Проверка диапазона (1-4)
- Проверка на пустой ввод
- Проверка формата ввода (`:` в строке)
- Проверка существования пользователя
- Понятные сообщения об ошибках

## 📝 Пример работы

```text

=============================================

Введите имя и после ":" друзей через запятую (или "стоп" для завершения):
анна: иван, петр, мария
иван: анна, петр, ольга
петр: анна, иван, мария
мария: анна, петр

Доступные пользователи: анна, иван, петр, мария, ольга

2. Найти взаимных друзей
3. Получить рекомендации
4. Выход

Введите первого пользователя: анна
Введите второго пользователя: иван
```
