# Проект 18: Банковский счёт (инкапсуляция) 💰

Консольное приложение для управления банковским счетом.  
Демонстрирует принципы инкапсуляции: приватные атрибуты, геттеры, контролируемый доступ через методы.

## 🎯 Что закрепляет

- Инкапсуляция (приватные атрибуты `__balance`, `__owner`)
- Геттеры через `@property`
- Методы для безопасного изменения состояния
- Обработка ошибок через `raise ValueError`
- Интерактивное меню с валидацией ввода

## 📋 Функционал

### 1. Класс `BankAccount`
- **Приватные атрибуты:**
  - `__owner` — владелец счёта (задаётся при создании)
  - `__balance` — текущий баланс (начальный 0)
- **Геттеры:**
  - `balance` — возвращает баланс (только чтение)
  - `owner` — возвращает владельца
- **Методы:**
  - `deposit(amount)` — пополнение счёта (проверка: amount > 0)
  - `withdraw(amount)` — снятие средств (проверка: amount > 0, достаточно средств)

### 2. Интерактивное меню
- Ввод имени владельца (защита от пустого ввода)
- Цикл с действиями:
  - Пополнить счёт
  - Снять деньги
  - Показать баланс (выводится всегда)
  - Выход

## 🛡️ Обработка ошибок

- **Ввод имени:** пустая строка → повторный запрос
- **Ввод номера действия:** буквы, число вне 1–3 → сообщение об ошибке
- **Ввод суммы:** буквы, ноль, отрицательное число → сообщение
- **Снятие больше баланса:** `ValueError` с сообщением
- Разделение ошибок: понятные сообщения для каждого случая
