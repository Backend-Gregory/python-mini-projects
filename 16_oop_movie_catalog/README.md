# Проект 16: Каталог фильмов с рекомендациями 🎬

Консольное приложение для управления фильмами, отслеживания просмотров и получения персональных рекомендаций.  
Работает с реальными данными 5000+ фильмов из открытого датасета TMDB.

## 🎯 Что закрепляет

- ООП (классы, методы, `@staticmethod`)
- Работа с JSON (загрузка, сохранение, преобразование)
- Использование `Counter` для подсчёта жанров
- Поиск по словарю (O(1))
- Функции для преобразования данных (объекты ↔ строки)
- Интерактивное меню с обработкой ошибок

## 📋 Функционал

### 1. Загрузка данных
- Фильмы загружаются из `movies.json` (5000+ записей)
- Жанры парсятся из строки JSON, рейтинг преобразуется в число
- Создаётся словарь `movie_by_title` для быстрого поиска

### 2. Пользователи
- Вход по имени (новый пользователь создаётся автоматически)
- История просмотров сохраняется в `users.json`
- При повторном входе история загружается

### 3. Добавление просмотров
- Поиск фильма по названию
- Оценка от 1 до 10
- Сохранение в историю пользователя

### 4. Рекомендации
- **По рейтингу** — топ-3 фильма с самым высоким рейтингом
- **По жанру** — на основе любимого жанра пользователя
- Комбинация результатов без дубликатов

## 🛡️ Обработка ошибок

- **Файлы:** защита от отсутствия, повреждения
- **Ввод:** пустое название, буквы вместо чисел, диапазон 1–10
- **Поиск:** фильм не найден — сообщение
- **Данные:** если нет просмотров — рекомендации не запрашиваются
