# Проект 14: Анализатор логов сервера 📊

Консольное приложение для анализа лог‑файлов веб‑сервера (формат access.log).  
Собирает статистику по запросам, IP‑адресам, статусам и объёму данных.

## 🎯 Что закрепляет

- Работа с реальными лог‑файлами
- `Counter` из `collections` (подсчёт частоты)
- `filter`, `lambda`, `max` с `key`
- `enumerate`, `most_common`
- Функции для перевода единиц измерения
- Обработка ошибок (файл, парсинг, деление на ноль)
- Один проход по файлу (оптимизация)

## 📋 Функционал

### 1. Общая статистика
- Общее количество запросов
- Количество уникальных IP‑адресов

### 2. Топ‑5 IP по количеству запросов
- Сортировка через `most_common`

### 3. Статистика по HTTP‑статусам
- ✅ Успешные (200)
- ⚠️ Перенаправления (301, 302)
- ❌ Ошибки клиента (404, 403)
- 💥 Ошибки сервера (500)

### 4. Объём переданных данных
- Общий объём в байтах / КБ / МБ
- Средний размер ответа

### 5. Самая частая ошибка
- Поиск через `filter` + `max`

### 6. Дополнительные проверки
- 📈 Есть ли запросы > 1 MB
- 🌍 Все ли запросы от локальных IP

## 🛡️ Обработка ошибок

- **Файл не найден / нет прав** — понятное сообщение, выход
- **Кривая строка в логе** — пропуск, программа не падает
- **Пустой файл** — защита от деления на ноль
- **Нет ошибок 4xx/5xx** — вывод «нет ошибок»
