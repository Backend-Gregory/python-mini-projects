# Проект 22: Генератор отчётов по продажам 📊

Консольное приложение для учёта продаж и формирования отчётов.  
Демонстрирует использование **генераторов** для поточной обработки данных.

## 🎯 Что закрепляет

- Генераторы (`yield`, бесконечные последовательности, фильтрация)
- Работа с файлами (чтение построчно, запись)
- Обработка дат (`datetime`, `strptime`, `strftime`)
- Валидация пользовательского ввода
- Интерактивное меню

## 📋 Функционал

### Класс `Sale`
- атрибуты: `date`, `product`, `category`, `amount`
- метод `__str__` — красивое представление

### Генераторы
- `read_lines(filename)` — читает файл построчно, выдаёт объекты `Sale`
- `filter_sales(sales, start, end, category)` — фильтрует продажи по дате и категории

### Интерактивное меню
1. **Добавить продажу** — ввод даты, товара, категории, суммы
2. **Показать отчёт** — фильтр по дате и категории, вывод списка и итогов
3. **Выход** — завершение программы

### Валидация
- Даты: формат ГГГГ-ММ-ДД, Enter — сегодня
- Сумма: число
- Товар и категория: не пустые
- Категория в отчёте: Enter — все категории

## 🛡️ Обработка ошибок

- **Ввод числа:** защита от букв
- **Ввод даты:** защита от неверного формата
- **Ввод суммы:** защита от букв
- **Файл:** защита от отсутствия, кривых строк
